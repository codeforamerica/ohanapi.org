Ohana API—An open directory of services

Out of the box, the Ohana API project provides an API with an admin interface for updating the data, and a website for searching the data.

  • ★ Free and open source
  • ★ Targeted search for human services
  • ★ Centralized data

OpenReferral

Ohana is the free / open source 'reference implementation' for the Open Referral Initiative, which is developing a new exchange format for community resource data. Open Referral is:

  • ★ An open standard, shaped by participatory community input.
  • ★ Interoperable with conventional human service directory systems.
  • ★ Optimized for indexing and retrieval by major search engines.

The Ohana platform can help you quickly demonstrate the value of opening community resource data in a standardized format. Go to OpenReferral.org to learn more about the Open Referral Initiative.
